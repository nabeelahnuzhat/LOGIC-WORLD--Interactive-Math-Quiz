# 🌟 Logic World - Math Quiz

An interactive mathematics quiz web application for elementary students.
Built with: HTML, CSS, JavaScript

## About
I independently developed Logic World, an interactive mathematics quiz 
web application for elementary students using HTML, CSS, and JavaScript. 
This is the first version of my ongoing goal to build a full-featured 
adaptive learning platform for young students. The application will support 
both Bangla and English to serve Bangladeshi elementary students while 
remaining accessible to international users.

## Current Features
- Student information form
- Embedded video lesson before quiz
- Multiple choice questions with A B C D options
- Score tracking and result screen

## Future Plans
- Bangla and English language toggle
- Separate question sets for Grade 3, 4, and 5
- Student progress tracking
- Mobile friendly design
- Teacher dashboard
